메모리
=============

예제
----------
```c
#include <stdio.h>

main()
{
   char c = 0;
   int i  = 0;
   flat f = 0;

   printf("c 메모리주소: %p\n", &c);
   printf("i 메모리주소: %p\n", &i);
   printf("f 메모리주소: %p\n", &f);
}
```

개념
---------
C언어에서 모든 변수들은 '메모리'에 저장됩니다. 
해당 변수의 크기만큼 메모리를 할당하고, 할당한 메모리의
맨 앞 주소를 내어줍니다.


#### &연산자
변수의 앞에 &연산자를 붙여주면 해당 변수의 메모리주소값을 얻을 수 있습니다.
