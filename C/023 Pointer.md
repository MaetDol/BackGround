포인터
===========
그래! 바로당신!

예제
----------
```c
#include <stdio.h>

void p(int n)
{
   printf("%d\n", n);
}

main()
{
   int a = 3;
   int b = 6;
   int c = a;

   int* pa = &a;
   int* pb = &b;

   p(a);
   p(b);
   p(c);
   p(*pa);
   p(*pb);

   *pa = 5;
   p(a);
   p(c);
}
```

문법
---------
022 Memory에서 변수들은 메모리를 할당하여 공간을 배정받고, 주소를 가지고 있다고 했습니다. 
포인터는 그 주소를 담는 변수입니다. 위 예제에서 pa는 a의 주소를 담는 역활을 하고 있습니다. 
a변수와 pa변수는 서로 다른 변수지만, 같은 메모리(값이 저장된 공간)를 바라보고 있으므로 값을 
공유한다고 할 수 있습니다. a에 5를 대입하면 pa도 5값을 가지게 되고, pa에 7을 대입하면 a또한 7 값을 가지게 됩니다.

#### 선언방법
        변수타입 * 변수명;
선언방법은 변수와 크게 다르지 않습니다. 
변수명 앞에 별을 다느냐 안다느냐의 차이입니다.

#### 엔드(&)연산자(번지 지정 연산자)
변수의 앞에 붙이면 해당 변수의 주소를 리턴합니다. 
배열, 문자열에는 사용할 수 없습니다. 
문자열은 문자로 이루어진 배열이므로 배열만 설명하겠습니다. 
배열명(int i[10] 에서 'i')은 배열이 있는 주소를 가르키는 포인터입니다. 
따라서 배열자체에는 사용이 불가능하지만 배열의 원소를 가르키는(i[1], i[3]...)방법을 사용하면
번지 지정 연산자를 사용할 수 있습니다.

#### 별표 연산자(포인터 연산자)
포인터를 선언할때도 사용되지만 포인터 값을 수정할 때도 사용됩니다. 
포인터 연산자를 사용하지않고 포인터 변수값을 조정하면 포인터 변수가 가지고 있는 주소가 바뀌지만 
포인터 연산자를 사용하고 값을 변경하면 해당 주소를 참조하여 메모리에 저장되어있는 값을 변경하게 됩니다.

#### 문자열 포인터
전에 한번, 문자열은 문자들로 이루어진 배열이라 했던 적이 있습니다.
        char* pts = "ABCD";
쌍따옴표로 감싸져있는 문자열 상수는 문자열이 변수에 대입되는게 아니라 메모리에 저장된 문자열을 
가르키는 포인터를 반환합니다.
