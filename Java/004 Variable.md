변수
============

예제
--------
```java
int i;
int k = 3;
char c = 'a';
float PI = 3.14f;
```

문법
--------
자바에서 변수는 '원시변수'와 '객체 레퍼런스', 두가지로 나뉩니다. 

### 원시변수
###### 부울, 문자
- boolean | JVM에 따라 다름 | true, false
- char | 16비트 | 0~65535

###### 정수
- byte | 8비트 | -128~127
- short | 16비트 | -32768~32767
- int | 32비트 | -2147483648~2147483647
- long | 64비트 | -짱짱큰 값~짱짱큰 값

###### 부동소수점 소수
- float | 32비트(바뀔 수 있음)
- double | 64비트(바뀔 수 있음)

#### 객체 레퍼런스
에는 객체에 대한 레퍼런스가 들어갑니다. 

#### 선언
원시변수, 객체 레퍼런스모두 동일하게 적용됩니다. 
1. 변수에는 유형이 있어야 합니다.
2. 변수에는 이름이 있어야 합니다.

#### 리터럴 값?
변수가 아닌 값입니다. 3.14, true, 77 등등...
```java
int i = 32 ;// 32가 리터럴 값
boolean b = false; // false가 리터럴 값
```

#### 변수명
- 알파벳, 밑줄, 달러기호($), 숫자를 조합하여 변수명을 명명할 수 있습니다.
- 알파벳, 밑줄, 달러기호($)중 하나로 시작해야합니다. 숫자로시작하면 안됩니다.
- 예약어는 사용할 수 없습니다(public, int, if ...).

#### 주의사항
* 자바에서 소수점이 들어있는것은 기본적으로 double로 간주하기 때문에 float값으로 쓰고싶다면 맨뒤에 'f'를 붙여야 합니다.
        float f = 3.14f;
* 원시변수에서 작은 크기의 변수값을 큰 크기의 변수로 줄수있지만 그 반대는 안됩니다. 정수 변수, 부동소수점 변수, 부울, 문자끼리도 불가능합니다.
        byte b = 3;
        int a = b; // 가능
        // a = b; 컴파일에러!
